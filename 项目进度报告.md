## 🎯 核心架构

### 插件系统 (Phase 1)
```
Plugin SDK
    ↓
Plugin Loader → Plugin Registry
    ↓
Event Bus ← Config Manager
    ↓
Plugin Store (Zustand)
```

### 命令系统 (Phase 2)
```
Command Registry
    ↓
Command Executor → Command History
    ↓
Shortcut Manager
    ↓
Command Store (Zustand)
    ↓
Command Palette UI
```

### 集成架构
```
React App
    ↓
├── Plugin System (Phase 1)
│   └── Plugins can register commands
│
├── Command System (Phase 2)
│   └── Commands can use plugins
│
├── State Management (Phase 3) ⏳
├── Config System (Phase 4) ⏳
└── UI Extensions (Phase 5) ⏳
```


### 3. 测试功能

**测试插件系统**:
1. 点击左侧 🧩 Plugins 图标
2. 点击 "Load Example Plugin"
3. 查看插件加载状态

**测试命令系统**:
1. 按 `Ctrl+Shift+P` 打开命令面板
2. 搜索命令（如 "save", "view"）
3. 选择并执行命令
