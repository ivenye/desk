## ❌ 如果遇到问题

### 问题 1: 依赖错误
```
Cannot find module 'zod'
```
**解决**: 运行 `npm install zod eventemitter3`

### 问题 2: 编译错误
```
Cannot find module '@/plugin-sdk'
```
**解决**: 检查 `tsconfig.json` 的 paths 配置

### 问题 3: 插件加载失败
```
Failed to load plugin
```
**解决**: 
- 检查插件路径：`../plugins/example-plugin/index.ts`
- 确认 plugins 目录存在
- 查看控制台详细错误

### 问题 4: Tauri 命令错误
```
Command not found: load_plugin_config
```
**解决**: 
- 重启开发服务器
- 确认 main.rs 中已注册命令
