# OpenClaw Studio - и®ҫи®Ўж–ҮжЎЈ

## йЎ№зӣ®жҰӮиҝ°

OpenClaw Studio жҳҜдёҖдёӘйӣҶжҲҗ AI д»Јз ҒеҠ©жүӢгҖҒеҸҜи§ҶеҢ–жҺ§еҲ¶йқўжқҝгҖҒеӨҡ Agent еҚҸдҪңзҡ„жЎҢйқўеә”з”ЁпјҢз»“еҗҲдәҶ Claude Code зҡ„з»Ҳз«Ҝ UI жһ¶жһ„е’Ң OpenClaw зҡ„ејәеӨ§иғҪеҠӣгҖӮ

## ж ёеҝғзү№жҖ§

### 1. AI д»Јз ҒеҠ©жүӢ
- **жҷәиғҪиЎҘе…Ё**пјҡеҹәдәҺдёҠдёӢж–Үзҡ„д»Јз Ғе»әи®®
- **д»Јз Ғе®ЎжҹҘ**пјҡиҮӘеҠЁжЈҖжөӢжҪңеңЁй—®йўҳ
- **йҮҚжһ„е»әи®®**пјҡдјҳеҢ–д»Јз Ғз»“жһ„
- **з»Ҳз«ҜйӣҶжҲҗ**пјҡзӣҙжҺҘжү§иЎҢе‘Ҫд»Өе№¶йў„и§Ҳз»“жһң

### 2. еҸҜи§ҶеҢ–жҺ§еҲ¶йқўжқҝ
- **Session з®ЎзҗҶ**пјҡжҹҘзңӢжүҖжңүжҙ»и·ғдјҡиҜқ
- **Agent зј–жҺ’**пјҡеҸҜи§ҶеҢ–еҲӣе»әе’Ңз®ЎзҗҶ sub-agents
- **е®һж—¶ж—Ҙеҝ—**пјҡжөҒејҸжҹҘзңӢжү§иЎҢж—Ҙеҝ—
- **Memory жөҸи§Ҳ**пјҡжҗңзҙўе’Ңзј–иҫ‘ MEMORY.md

### 3. еӨҡ Agent еҚҸдҪң
- **е·ҘдҪңжөҒзј–иҫ‘еҷЁ**пјҡжӢ–жӢҪејҸи®ҫи®Ў Agent еҚҸдҪңжөҒзЁӢ
- **ж¶ҲжҒҜи·Ҝз”ұ**пјҡAgent й—ҙйҖҡдҝЎеҸҜи§ҶеҢ–
- **д»»еҠЎеҲҶеҸ‘**пјҡиҮӘеҠЁеҲҶй…Қд»»еҠЎз»ҷжңҖеҗҲйҖӮзҡ„ Agent
- **з»“жһңиҒҡеҗҲ**пјҡжұҮжҖ»еӨҡдёӘ Agent зҡ„иҫ“еҮә

### 4. йЎ№зӣ®з®ЎзҗҶ
- **ж–Үд»¶жөҸи§ҲеҷЁ**пјҡж ‘еҪўз»“жһ„жҹҘзңӢйЎ№зӣ®ж–Үд»¶
- **Git йӣҶжҲҗ**пјҡжҹҘзңӢ diffгҖҒжҸҗдәӨеҺҶеҸІ
- **д»»еҠЎзңӢжқҝ**пјҡKanban йЈҺж јзҡ„д»»еҠЎз®ЎзҗҶ
- **еҝ«йҖҹжҗңзҙў**пјҡе…ЁеұҖж–Үд»¶е’ҢеҶ…е®№жҗңзҙў

```typescript
// src/components/terminal/Terminal.tsx
export function Terminal() {
  const terminalRef = useRef<HTMLDivElement>(null);
  const xtermRef = useRef<XTerm>();
  
  useEffect(() => {
    const term = new XTerm({
      theme: { background: '#1e1e1e' },
      fontSize: 14,
      fontFamily: 'JetBrains Mono',
    });
    
    term.open(terminalRef.current!);
    xtermRef.current = term;
    
    // иҝһжҺҘеҲ° OpenClaw exec
    term.onData((data) => {
      invoke('exec_command', { input: data });
    });
  }, []);
  
  return <div ref={terminalRef} className="h-full" />;
}
```
