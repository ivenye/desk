# 🚀 快速测试 - 3 步开始

## 最简单的测试方法

### Step 1: 安装依赖（5 分钟）

```bash
cd "C:\Users\Administrator.DESKTOP-E9TDD9I\Desktop\claude test\desk-main-project"

# 安装新增依赖
npm install zod eventemitter3

# 如果缺少其他依赖
npm install
```

### Step 2: 启动应用（1 分钟）

```bash
# 启动 Web 开发模式
npm run dev
```

然后在浏览器打开：`http://localhost:5173`

### Step 3: 测试核心功能（10 分钟）

#### ✅ 测试 1: 命令面板（最重要）
1. 按 **Ctrl+Shift+P**
2. 看到命令面板弹出 ✅
3. 输入 "file" 搜索命令 ✅

#### ✅ 测试 2: 插件系统
1. 点击侧边栏 **🧩** 图标
2. 看到插件管理界面 ✅
3. 点击 "Load" 加载示例插件 ✅

#### ✅ 测试 3: 主题切换
1. 打开命令面板（Ctrl+Shift+P）
2. 搜索 "theme"
3. 切换主题，看到界面颜色变化 ✅

## 🎯 如果遇到问题

### 问题 1: npm install 失败
**解决**: 检查网络，或使用国内镜像
```bash
npm config set registry https://registry.npmmirror.com
npm install
```

### 问题 2: TypeScript 报错
**解决**: 先忽略，直接运行 `npm run dev`

### 问题 3: 应用无法启动
**解决**: 
1. 检查端口 5173 是否被占用
2. 查看终端错误信息
3. 把错误信息告诉我
