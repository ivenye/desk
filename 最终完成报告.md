### 数据流

```
用户操作
    ↓
UI 组件
    ↓
Root Store (Zustand)
    ↓
├── Command System → Command Registry → Execute
├── Plugin System → Plugin Loader → Register
└── State Updates → Middleware → Persist
    ↓
Tauri Backend (Rust)
    ↓
文件系统 / 配置 / 插件
```
