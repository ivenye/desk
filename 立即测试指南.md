## ❌ 常见问题

### 问题 1: npm 命令未找到

**错误**:
```
'npm' is not recognized as an internal or external command
```

**解决**:
1. 安装 Node.js: https://nodejs.org/
2. 下载 LTS 版本
3. 安装后重启终端

### 问题 2: 依赖安装失败

**错误**:
```
npm ERR! code ENOENT
```

**解决**:
```bash
# 删除 node_modules 和 package-lock.json
rm -rf node_modules package-lock.json

# 重新安装
npm install
npm install zod eventemitter3
```

### 问题 3: Tauri 编译失败

**错误**:
```
error: could not compile `desk`
```

**解决**:
1. 确保安装了 Rust: https://rustup.rs/
2. 更新 Rust: `rustup update`
3. 重新运行: `npm run tauri:dev`

### 问题 4: 插件加载失败

**错误**: 点击 "Load Example Plugin" 后显示错误

**解决**:
1. 检查 `plugins/example-plugin/index.ts` 是否存在
2. 查看控制台详细错误信息
3. 确认路径正确：`../plugins/example-plugin/index.ts`

### 问题 5: TypeScript 编译错误

**错误**:
```
Cannot find module '@/plugin-sdk'
```

**解决**:
检查 `tsconfig.json` 中的 paths 配置：
```json
{
  "compilerOptions": {
    "paths": {
      "@/*": ["./src/*"]
    }
  }
}
```
